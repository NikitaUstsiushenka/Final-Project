На данном этапе проект еще не завершен полностью. Полноценно реализована роль "Гость", работа ведется над
клиентом и админом. В данном проекте максимально старался использовать паттерны, где они нужны. Для сборки
проекта требуется подгрузить .jar файлы: javax.servlet.jsp.jstl-1.2.1, javax.servlet.jsp.jstl-api-1.2.1,
log4j-api-2.11.0, log4j-core-2.11.0, mysql-connector-java-5.1.45-bin. В папке sql предоставлены все скрипты для 
сборки БД. Для запуска проекта требуется скопировать папку web и залить ее к себе на сервер.
В дальнейшем мой проект будет залит вот на этот аккаунт GiHub: https://github.com/NikitaUstyushenko
